Reto #9


Desarrollar un algoritmo y ponerlo en funcionamiento que simule el juego de adivinar el número.
El algoritmo debe generar un número aleatorio entre 0 y 1000 y deberás ingresar números hasta que des con el número secreto

Solución

import random
num_intentos=0
num_min=0
num_max=1000
num_oculto=random.randint(num_min, num_max)

print("¡Hola, vamos a jugar! ¿Puedes adivinar un número del 0 al 1000?")

while num_intentos <1000:
print("Ingresa tu número: ")
intento= input()
intento=int(intento)
if num_oculto < intento:
print("No es el número misterioso. ¡Sigue intentando! \n\
El número misterioso es menor que el número que ingresaste")

if num_oculto > intento:
print("No es el número misterioso. ¡Sigue intentando! \n\
El número misterioso es mayor que el número que ingresaste")

if num_oculto == intento:
print("¡Ganaste! Este es el número correcto.")

break
